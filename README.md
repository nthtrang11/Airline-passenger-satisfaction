# Airline passenger satisfaction Analysis Dashboard

Mô tả dự án:
- Dự án này phân tích dữ liệu sự hài lòng của hành khách hàng không (Airline Passenger Satisfaction) dựa trên bộ dữ liệu từ Kaggle với hơn 104K records. Sử dụng Power BI, tôi đã xây dựng một dashboard tương tác để khám phá các yếu tố ảnh hưởng đến satisfaction, phân khúc khách hàng, và đưa ra khuyến nghị kinh doanh.
- Mục tiêu chính:
+ Xác định key influencers (ví dụ: Inflight WiFi service >4 tăng satisfaction lên 62%).
+ Visualize phân phối satisfaction theo demographics (giới tính, tuổi, class, travel type).
+ Đánh giá tác động của delay và service ratings đến loyalty.

+ Link dataset: https://www.kaggle.com/datasets/teejmahal20/airline-passenger-satisfaction
+ Link dashboard: https://app.powerbi.com/reportEmbed?reportId=cfbcb953-8084-44f6-9ac7-11fc657c9bef&autoAuth=true&ctid=2a141a9b-4ef4-4094-a1e5-59bf690777c6&actionBarEnabled=true&reportCopilotInEmbed=true
  
Công nghệ sử dụng: 
+ Power BI (DAX).
+ Python (preprocessing dữ liệu, phân tích dữ liệu).
+ Dữ Liệu: Kaggle, dataset: Airline Passenger Satisfaction.
